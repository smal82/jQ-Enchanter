== Description ==
Un plugin per aggiungere effetti jQuery personalizzati. Puoi attivare anche pi√π effetti in contemporanea.

Effetti attualmente presenti nella versione 1.0.4:
1. Neve: un semplice effetto per far nevicare sul sito e far accumulare la neve in basso, mettendo anche negli angoli dello schermo un paio di pupazzetti;
2. Background-changer: puoi cambiare lo sfondo del sito in modo casuale con delle immagini presenti nel plugin;
3. Babbo Natale: fai passare Babbo Natale nella parte bassa dello schermo con un movimento ondulatorio;
4. Befana: fai passare la Befana nella parte alta dello schermo con un movimento ondulatorio;
5. Karaoke: fai scorrere il testo della canzone "Santa Clause is coming to town" con un effetto karaoke;
6. San Valentino: una semplice parola "Love" con un cuoricino da far comparire il giorno di San Valentino.


Altri effetti saranno inseriti nelle prossime versioni in elaborazione.

1. Scarica il plugin
2. Caricalo su WordPress
3. Attivalo
